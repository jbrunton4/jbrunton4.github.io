<HTML Tags>

<html> - Encloses the HTML document.
<head> - Encloses the header.
<title> - Encloses the HTML document's title.
<meta> - Mainly used for setting charset e.g. <meta charset='utf-8'>.
<link> - Mainly used for linking a stylesheet using <link rel='stylesheet' href='style.css'>.
<style> - Used for internal CSS.
<body> - Encloses the main part.
<div> - Makes a division or container.
<img> - Adds an image using an src e.g. <img src='myimage.png'>. Can also take width and height.
<video> - Similar to image, but for videos. Also takes controls, autoplay and mute.
<li> - Encloses a bullet point.
<b> - Makes text bold.
<i> - Makes text italic.
<u> - Makes text underlined.
<s> - Makes text strikethrough.
<br> - Breaks to a new line.
<p> - Encloses a paragraph.
<a> - Makes a link using a href e.g. <a href='https://www.google.com'>.
<select> - Makes a dropdown menu.
<option> - Makes an option within a dropdown menu. Takes a value.
<input> - Makes an input. Type can be button, text, etc.
<font> - Edits the font's color, size, etc..
<script> - Encloses internal JavaScript. Can also use an src to link external.
<!-- --> - Encloses a comment, code that is not shown or run.
<abbr> - Defines an abbreviation.
<footer> - Encloses the footer.
