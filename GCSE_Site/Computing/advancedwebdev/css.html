.CSS:Properties

Color: defines text color.
Background: defines background color.
Border: defines border using width, line style, color.
Margin: defines margin width/height.
Margin-top/bottom/left/right: defines individual margin width/height.
Padding: defines padding width/height.
Padding-top/bottom/left/right: defines individual padding width/height.
Width: defines width.
Height: Defines height.
Min/max-width/height: Defines minimum/maximum width/height.
Transform: change appearance in size/orientation/position using functions such as translate(xpx,ypx), rotate(xdeg) or scale(x).
Transition: defines time taken to transform.
Animation-name: defines animation name.
Animation-duration: defines how long the animation takes.
Animation-delay: defines how long after loading animation should play.
Animation-iteration-count: defines how many times animation should play.
Animation-direction: defines in which direction animation should play.
Animation-fill-mode: defines in which direction animation should extend.
Font-size: defines size of font.
Font-weight: Defines boldness.
Font-family: defines the family of font (generally how it looks).
