<JavaScript>

JavaScript is a client-side programming language (which means it runs on the user's computer and not where the website is stored). It's how a user interacts with a page, for example when you click the "Log In" button.

The most simple way for a user to cause some JavaScript to be run is with a button. You make buttons in HTML like this: <input type='button' value='Click me!' onclick='myFunction();'>. This would use the function myFunction() , if you used a different name then just replace myFunction() with the name of your function. Here is an example with no CSS:

There are 2 ways to do JavaScript: internal and external. Internal JavaScript is where JavaScript is written inside the HTML page, between <script> tags. You can also make a different file (like you would with CSS) with the .js file type, and just add an src to your script tag like so: <script src='myscript.js'>.

You'll need to know how to declare a function. It's simple really: just type function myFunction() { /* This is the function */ }.

A simple function in JavaScript is alert(), which makes an alert like . Inside the brackets, you put a message, like this: alert("Hello, world!"). So, here's an example:
<script>
....function sayHi() {
....alert("Hello, world!");
</script>


By the way, always use a semicolon after the end of a statement like alert();!

You can make a variable (data stored as a word) using var, like so:
var myNumber;
myNumber = 123;
You can also shorthand this:
var myNumber = 123;
This can be an integer (whole number), real number (non-integer number), string (text) or even an array (list). Arrays are made like so:
var myArray = ["hi", 1, 2.3];
You can use these words in the place of strings or numbers, as long as they have been declared.

JavaScript isn't like Python - errors don't stop the program altogether. Instead, errors are ignored and logged to the console (normally accessed by pressing F12 and clicking on console).
