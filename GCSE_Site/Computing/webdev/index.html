HTML
HTML (HyperText Markup Language) is the essential language of the web. You can't have a website without it. This page, every other page on this site, and any page that you can ever visit, was programmed using HTML. You can announce that something is a HTML document by typing <!DOCTYPE html> at the top of a document. The entire HTML document should be inside the <html></html> (putting a tag with a / before the word will close or end the tag).

For the header of a basic HTML page, you need a <head></head> tag. Inside of this, you may also want to add a <title>My Page</title> for a page title. You don't need to, but to avoid errors in the console, you might also want to add <meta charset='utf-8'> .

The main part of a page should go inside a <body></body> tag. Some useful tags in here are <br></br> to start new lines, and <img src='myimage.png'></img> for images (replace img with video for videos).

Example:
<!DOCTYPE html>
<html>
....<head>
........<title>My Webpage</title>
........<meta charset='utf-8'>
........My Webpage
....</head>
....<body>
........This is my webpage<br>
........<img src='myimage.png'></img>
....</body>
</html>
