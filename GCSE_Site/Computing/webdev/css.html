CSS
CSS (Cascading stylesheet) is a way of decorating a HTML page.
To start, you'll need to know how to say what you want to effect. For a normal tag e.g. <img> you just say img, for something with an id e.g. <div id="imagediv"> , you will need to place a hashtag before the name e.g. #imagediv. Finally, an object with a class e.g. <div class="mydiv"> will need a dot before it e.g. .mydiv (you can also reference elements inside elements e.g. .mydiv #imagediv). Some common CSS properties are width, height, background, and color.
#example {
....width: 400px;
....height: auto;
....max-width: 100%;
....background: powderblue;
....color: black;
}
