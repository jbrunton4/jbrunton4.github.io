JavaScript
JavaScript is a method of interactinhg with a webpage. There are 2 simple things you can do with javascript: program a button or make a list. First, though, you need to learn the HTML: <input type='button' value='click me' onclick='myFunction();'> makes a button (that has the function myFunction();) and making dropdown menus like this one: You make this using a <select id='myselect'> tags with options like <option value='eggs'>Eggs</option> (the value in JS is this: document.getElementById('myselect').value;).

The simplest way to do JavaScript is internal, between <script> tags. To make an alert, you use alert('myalert');, and to make a function you do function myFunction() {alert('hi');} where your code goes between the {} tags.
