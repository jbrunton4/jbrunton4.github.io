Raster Images
Raster images are made from pixels (derrived from pixel+element), which are coloured squares on a screen. These are arranged in a way that makes them appear as an image. You can compress them 2 ways: Lossy, where pixels are removed (this makes for a smaller file) or lossless, where the image itself is shrunk (this makes for a larger file size but a better image once restored). There are many file types for images:
  • JPEG / Joint Photographic Experts Group(.jpg)
  • Gif / Graphics Interchange Format(.gif)
  • BMP / Bitmap (.bmp)
  • Tiff / Tagged Image File Format (.tiff)
  • PNG / Portable Network Graphic (.png)
