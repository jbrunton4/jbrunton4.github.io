Python Basics
Python is a beginner-friendly programming language which is relatively easy to understand. There's a small amount you'll need to know for GCSE computing. This page covers what you should have learnt in year 9.

If you don't already have python installed, you should install it here now. Once you do, you should have a program called IDLE - you can now create and run .py and .pyw files, as well as use pip (we can worry about that in advanced, and you probably won't be able to use it at school).

Let's start by making a "Hello world" program. A "Hello world" program simply shows the user a message, in this case it's "Hello, world!" or some variation of the two words. In python, to display a message to the user, we use print(), and put a message in quotes ("",'' or ``), like this: print("Hello, world!").

If you hit F5 (or Fn+F5) or go to run>run module, the program should start and finish just fine, but if you run it without IDLE or another IDE, it will just close straight away. This is our first use for the input() function. There are 2 uses, the second of which we will come across later, but for now we will use it to stop the program until the user presses enter. Putting input at the end of your code will stop it from closing automatically. Now we have 2 lines of code:
print("Hello, world!")
input()
Try saving this code as "my first program.py" and running it. See what happens!

Variables are a useful feature - they allow data to be stored in keywords, for example, if I didn't want to type "supercalifragalisticexpialidocious" more than once, I could store it in a variable like so: myVar = "supercalifragalisticexpialidocious". Variables can be many different types of data, but you only really need to worry about strings (str), integers (int) and real numbers (float). A string is some text, e.g. "Hello, world!" from earlier. An integer, or int, is a whole number, like 5 or 420. A real number, or float as they are known in python, is a decimal number, or a non-integer number. It is any number that is real, finite and rational, for example, pi is not a float as it is irrational, infinity is not a float as it is not finite, and the square root of -1 is not a float because it is not a real number.

You can encounter some problems with variable types. Try this:
myNum = 2
print(myNum)
What do you get? It's an error, isn't it? That's because you can only print strings. You can solve this one of 3 ways:
1) Making the variable as a string: myNum = "2" This will stop any maths from being done to it, but you can join it to other strings, get a character from a position, or print it.
2) Turning the variable into a string: After you've defined the variable, you can do myNum = str(myNum) or, even shorter, just str(myNum).
3) Printing the string, but keeping the variable as an int. You do this like so: print(str(myVar)).
. You can change variable types around using int(), float() or str().

Remember that second use for input() that we talked about earlier? It's time for that. You can get a user to give you a variable using the input function - it works similarly to print, in that it will take a string between the brackets and display this to the user. Let's try getting a user's age:
age = input("How old are you?")
By default, an input is a string. There are 3 ways to convert it to an int: 2 of which you've already seen (int(age) or age = int(age)), or put your entire input inside an int function like this: age = int(input("How old are you?")).

With ints and floats, you can do maths. There are 11 things you can do to define a new variable using maths:
  • + adds 2 variables.
  • - subtracts one variable from another.
  • * multiplies 2 variables.
  • / divides one variable by another.
  • +=, -=, *= and /= mean itself plus/take/times/divide by something else.
  • = makes a variable equal to something else.
  • ** means to the power of.
  • // means remainder after division.


    • You can also compare 2 variables in an if statement. There are 6 more things you need to know for if statements:
  • == means is equal to (in an if statement).
  • != means not equal to (in an if statement).
  • > means greater than.
  • < means smaller than.
  • >= means greater than or euqal to.
  • <= means smaller than or queal to.


    • An if statement means if (x) is true then do (y). Say I have an integer called age, and I wanted to check it it's over 18, I would do this:
    if age >= 18:
    ....print("Age is over 18")
    else:
    ....print("Age is under 18")
