Python Advanced
Modules are helpful in python as they allow you to do things that you cannot do in regular python. Python comes with a list of pre-installed modules such as random, turtle, tkinter, and subprocess among others. There are also modules that you can install, for example, domino's pizza has a module known as pizzapy, which allows pizza to be ordered through a python program. Once you set up pip (look for a tutorial online), you can type pip install modulename. For example, if you wanted to install pyGame, you would do it like so:

For now, you don't need to worry about installing modules, however, if you're not sure if the computer that you want your program to run on has the required module, you can do this:
from subprocess import call
call("pip install pygame")
Subprocess' call causes a command to happen (if it can), so it's usefull for automatically installing modules via pip.

A good first module to learn is random. You import it using import random. If you don't want to type randint. before everything to do with the module, simply type from random import *. The random module includes 2 very useful functions: random.choice(), which takes multiple values or variables and randomly chooses between them, and random.randint(), which takes 2 arguments (integers) and generates a random number between them, including the first but not the second.

OS is a module used for getting information or controlling the operating system, for example, os.getlogin() gets the user's login name, and os.startfile() opens the file in the brackets, e.g. os.startfile("notepad.exe").

Some modules, such as turtle, are more complex. You can use turtle to create drawings or even games. Start by importing the turtle module, then making a screen like this: wn = turtle.Screen() (it doesn't have to be called wn, you can give it any name you want). You can change the background colour using wn.bgcolor("black"), or whatever colour you wish. Remember to use the american spelling of the word colour (color).

A turtle is like a pen. To create a turtle, you need to give it a name, like this: myTurtle = turtle.Turtle() (remember to use a capital T in the second word "turtle"). By default, turtles have a speed of 1, a pensize of 1, are the colour black, have pen down, and are at (0,0), and facing right. For basic movement,] turtles can go forward(pixels) or fd(pixels), left(degrees) or lt(degrees), right(degrees) or rt(degrees). You can also set the heading, using seth(degrees).

FOR loops are useful in python - they allow repeats to happen, which is useful for making regular shapes. You create one like this:
for a in range(4):
....pass
You can use this to draw shapes like this:
import turtle
wn = turtle.Screen()
myTurtle = turtle.Turtle()
for a in range(4):
....myTurtle.fd(100)
....myTurtle.lt(90)
Some other features are turtle.circle(radius) and turtle.write(string).

There are many more modules with documentation and tutorials to be found online.

At some point, by your own fault or the fault of a user, your program is going to get some errors. Any error in python will stop the entire program from running - this is where error handling comes in. The code in "try" will be run, but if it fails, the code in "except" will be run. Here is an example:
myNum = input("Pick a number : ")
try:
....int(myNum)
except:
....print("Sorry, that's not a whole number.")
If you know exactly which type of error to expect, e.g. int('abc') would give a TypeError, you can put the error message in a variable like this:
try:
....int("abc")
except TypeError as myError:
....print(str(myError))
