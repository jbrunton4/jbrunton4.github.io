Scratch
Scratch is a GUI object-oriented programming language in which a user can drag and drop blocks of code like a jigsaw puzzle to easily create a program.

First, for something to happen, you will need an event - events make code run when something happens. There are 6 types:
When green flag clicked means when the user clicks on the 'Go' button above the program. When _____ key pressed means anytime a key is pressed on the keyboard. When this spire clicked is specific to sprites, and it means that when the object (sprite) that has this code is clicked, it will run. When backdrop switches to _____ means it will happen when the stage looks a certain way (the stage is the place where the sprites are). When _____ > _____ means that the code will run when the left variable is bigger than the right. When I receive _____ means that if itself or another sprite broadcasts a message (also in events), the code will run.

Control shows what should happen when and under what conditions, when something should stop and deals with cloning.
Wait _____ secs means that before the next line happens, this many seconds should pass. Repeat _____ means that what is inside it will repeat that many times. Forever means that it will repeat forever. If _____ then means what is inside will only happen if the condition is met. If _____ then, else means the same as if _____ then except what is in else will happen if not. Wait until _____ means the next code will not run until the condition is met - repeat until means the opposite, that it will not stop until the condition is met. Stop _____ means that the code will stop for what is selected. When I start as clone is an event for clones specifically. Create clone of _____ means that the object will be cloned, and delete this clone means that the clone will stop existing.
Sensing can be used, in part, in if statements and while loops. Touching _____ is a condition that means if the sprite is touching what is specified. Touching color _____ or color _____ is touching _____ means if the sprite or colour is touching the selected colour. Distance to _____ will make a number representing the distance from the sprite's origin to the specified object. Ask _____ and wait means that the sprite will say something and try to get an answer. If answer is ticked, the answer will be stored in the answer variable. Key _____ pressed means when a certain keyboard key is hit. Mouse down means while clicking. Mouse X and mouse Y are the X and Y positions of the mouse on the screen. Loudness is how loud something is. Video is just a video. Set video transparency means how see-through it is. Timer is a variable of how much time has passed, reset timer sets timer to 0. _____ of _____ is the property of another sprite. Current minute means how long since the program began in minutes. Days since 2000 means how many days have passed since 1/1/2000, and username is the user's name.

Operators are used to manipulate variables.
These are really simple - all you might not understand is * (times) and / (divide).

Motion makes sprites move across the stage.
Rotation style dictates on what axes the sprite can move - the rest are self-explanatory.

Looks shows how the sprite will look.
Say and think create speech and thought bubbles. Hide makes the sprite invisible, and show reverses this. Switch costume and backdrop will change how the sprite or stage look. Color effect is the tint of the sprite. Change size makes the sprite bigger or smaller. Go to front or go back _____ layers shows which sprite should be in front of which other.

The sound tab makes sound in the program, simple as that.
Play sound _____ (until done) plays a pre-existing sound from scratch. Stop all sounds stops them. Play drum _____ for _____ plays the selected drum sound for the specified amount of beats. Rest is like wait, but for beats. Play note plays a note from the western scale for the specified amount of beats. Set instrument chooses the instrument sound. Change volume and tempo changes how loud and fast it is respectively.

Pen makes sprites draw as they move across the screen.
Clear destroys all ink. Stamp makes a single point of ink where the sprite is. Pen down starts drawing and pen up stops drawing. Set color changes the colour of the ink - each has a number, which can be changed. Change shade and size change how light and thick the stroke is.

Variables and lists allow for the storage and manipulation of data.
